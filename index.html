Public Projects

  • Deep Metric Learning Baselines:

    PyTorch implementations for reference deep metric learning methods with standard/state-of-the art loss variants (e.g. Margin, ProxyNCA, N-Pair) and sampling options (semihard, distance). See this Deep Metric Learning Github Repo.

  • 2D Liver Lesion Segmentation:

    Implementations of various state-of-the-art 2D U-Net variants, using different network backends (e.g. dense, residual) and extensions to architecture (e.g. squeeze-and-excitation, pyramid-pool) and training (e.g. auxiliary inputs). See this LiTS Lesion Segmentation Github Repo.

  • Smaller Projects:

    Implementations of a web crawler for car images, car masking network for the Carvana Image Masking Challenge and a simulated self-driving car. See the Crawler Repo, Masking Repo and the Self-Driver Repo.
